# biblepay_rpc
Displays some basic stats from the Biblepay blockchain.

Usage:

biblepay_rpc.exe -u RpcUser -p RpcPassword -t

Requires an RPC username and password be set in biblepay.conf using the rpcuser=user and rpcpassword=pass options.

Depends on BitcoinLib (https://github.com/GeorgeKimionis/BitcoinLib) and CommandLineParser (https://github.com/gsscoder/commandline).
